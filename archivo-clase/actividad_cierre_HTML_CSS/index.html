Puzzle 2: Dashboard Grid Desordenado

Alta

Objetivo: Organizar el dashboard en una estructura de grid específica.

Instrucciones: Modifica .grid-dashboard y las clases de sus hijos para lograr este layout:

  • Header debe ocupar todo el ancho (2 columnas) en la primera fila
  • Sidebar ocupa la columna izquierda (2 filas de alto)
  • Main Content ocupa la posición central derecha
  • Stats ocupa la posición inferior derecha

Puzzle 4: Tarjetas Responsive

Medio

Objetivo: Crear un grid responsive que se adapte automáticamente.

Instrucciones: Modifica .responsive-grid para:

  • Usar CSS Grid con columnas que se ajusten automáticamente
  • Cada columna debe tener un mínimo de 250px y máximo de 1fr
  • Gap de 20px entre elementos
  • NO usar media queries (usa auto-fit o auto-fill con minmax)

Puzzle 5: Completar la Animación

Medio

Objetivo: Hacer que la tarjeta tenga una animación suave al hacer hover.

Instrucciones: Modifica .animated-card y .animated-card:hover para:

  • Agregar transition de 0.3s en todas las propiedades
  • Al hacer hover: transformar con scale(1.05)
  • Al hacer hover: agregar una sombra más pronunciada (box-shadow: 0 10px 30px rgba(0,0,0,0.3))
  • Al hacer hover: rotar levemente (rotate: 2deg)
