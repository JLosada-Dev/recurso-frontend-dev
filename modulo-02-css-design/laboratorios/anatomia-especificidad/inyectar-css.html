Métodos para Inyectar CSS en HTML

1. CSS Inline (En línea)

Este CSS se aplica directamente en el atributo "style" del elemento HTML. Tiene la mayor especificidad pero es difícil de mantener.

2. CSS Interno

Este CSS se define en la etiqueta <style> dentro del <head>. Se aplica solo a esta página HTML.

3. CSS Externo

Este CSS se carga desde un archivo externo usando <link>. Permite reutilizar estilos en múltiples páginas.

Ejemplo de Especificidad

Este div tiene tanto CSS interno como inline. El borde será rojo (inline) pero el fondo azul (interno).

Método Especificidad Ventajas Desventajas
Inline Muy Alta Rápido, específico Difícil mantener
Interno Media Una sola página No reutilizable
Externo Baja Reutilizable, organizado Archivo adicional
