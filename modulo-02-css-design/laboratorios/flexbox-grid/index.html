Cheat Sheet

Flexbox Properties

/* Container */ display: flex; flex-direction: row | column; justify-content: flex-start | center | flex-end | space-between | space-around; align-items: flex-start | center | flex-end | stretch; flex-wrap: nowrap | wrap; gap: 1rem; /* Items */ flex: 1; flex-grow: 1; flex-shrink: 0; flex-basis: auto; align-self: auto | flex-start | center | flex-end;

Grid Properties

/* Container */ display: grid; grid-template-columns: 1fr 2fr | repeat(3, 1fr) | auto 1fr; grid-template-rows: auto 1fr auto; grid-template-areas: "header header" "sidebar main"; gap: 1rem; justify-items: start | center | end | stretch; align-items: start | center | end | stretch; /* Items */ grid-column: 1 / 3; grid-row: 1 / 2; grid-area: header; justify-self: start | center | end | stretch; align-self: start | center | end | stretch;
