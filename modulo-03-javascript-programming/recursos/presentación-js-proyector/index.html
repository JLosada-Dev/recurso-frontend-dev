Unidad 3: Programación con JavaScript

Objetivo: Comprender los fundamentos de JavaScript, manipular el DOM y manejar eventos.

3.1 Fundamentos de JavaScript

Duración: 3 horas

  • Sintaxis básica y tipos de datos
  • Variables (var, let, const)
  • Operadores aritméticos y lógicos
  • Estructuras de control (if/else, bucles)
  • Funciones y scope

Sintaxis básica y tipos de datos

En JavaScript tenemos tipos primitivos como:

  • String - Texto
  • Number - Números
  • Boolean - true/false
  • Null y Undefined
// Ejemplo
let nombre = "Camilo";
let edad = 25;
let activo = true;

Variables: var, let y const

var: Ámbito global o de función.
let: Ámbito de bloque.
const: No se puede reasignar. 

// Ejemplo
var x = 10;
let y = 20;
const PI = 3.1416;

Operadores

Aritméticos: + - * / %

Lógicos: && || !

// Ejemplo
let suma = 5 + 3; // 8
let esMayor = (10 > 5) && (5 > 2); // true

Estructuras de control

if / else y bucles como for, while.

// Ejemplo
if (edad >= 18) {
  console.log("Es mayor de edad");
}

for (let i = 0; i < 3; i++) {
  console.log("Iteración " + i);
}

Funciones y Scope

Las funciones son bloques reutilizables. Ejemplo:

function saludar(nombre) {
  return "Hola " + nombre;
}
console.log(saludar("Ana"));

Scope: Define dónde son accesibles las variables.
